# Freelancer & Agency Roster

Internal directory of multimedia & creative partners for Foundant Technologies, with an AI-powered recommendation engine.

## Setup

1. `npm install`
2. `npm start` (local dev)
3. `npm run deploy` (deploy to GitHub Pages)

## API Key

The AI recommendation engine requires an Anthropic API key. Edit `src/FreelancerRoster.jsx` and paste your key on the `const apiKey = "..."` line.
